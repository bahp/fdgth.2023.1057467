ls2d.metrics.gmm_intersection_area(y_pred=None, y=None, gmm_matrix=None, normalize=False, **params)[source]

Compute the intersection area.

The method can receive either the two input parameters y_pred and y, or just the previously computed gmm_matrix.

Parameters

  • y_pred (np.ndarray) – The projections in the embedded space.

  • y (np.ndarray) – The labels to compute the GMMs.

  • gmm_matrix (np.ndarray) – The gmm intersection area matrix.

  • normalize (boolean) – If true the area will be normalised; that is, will be divided by the total area of the ellipsoids (sum of the gmm intersection matrix diagonal)

Returns

area – The sum of all the intersection areas.

Return type

float

class ls2d.pipeline.PipelineMemory(steps, memory_path=None, memory_mode='pickle', **kwargs)[source]

Bases: sklearn.pipeline.Pipeline

This is an extension of the scikits pipeline.

The only aim of this extension is to allow to save each of the models in memory just after they have been fit. To do that it uses a global variable to track what algorithm is being tested and what split is in.
